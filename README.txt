Essa versão do código foi feita somente para o protótipo do GREG LEO (SHIELD do GREG)

O primeiro protótipo feito manualmente não possui as teclas/pinos:

'f' // Header1(f) // Pino 4
'd' // Header2(d) // Pino 3
's' // Header3(s) // Pino 2
'a' // Header4(a) // Pino 1
'w' // Header5(w) // Pino 0

Logo, para que os pinos associados não interfiram no funcionamento do protótipo eles foram colocados como comentários no código.

Qualquer duvida, consultar a engenharia.
